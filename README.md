# job4j_cinema

## О проекте
Сервис по покупке билетов в кинотеатр.

***

##Навигационная панель:

<ol>
    <li>Лого.</li>
    При клике на него выполняется переход на главную страницу;
    <li>Расписание.</li>
    Выводятся сеансы и связанные с ними фильмы;
    <li>Кинотека.</li>
    Выводится список фильмов, которые показываются в кинотеатре;
    <li>Регистрация/Вход</li> 
    Если пользователь не вошел в систему
    <li>Имя пользователя/Выйти</li>
    Если пользователь вошел в систему.
</ol>

***

##Разделение прав:

Все пользователю имеют право просматривать информацию на сайте;
Только зарегистрированные пользователю могут покупать билеты. 
Если пользователь не зарегистрирован и нажимает на кнопку "Купить", 
то его перебрасывает на страницу входа.
